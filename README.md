## Features

- `游릭` Syntax Highlighting
- `游릭` Auto completions
- `游릭` Auto Indentation
- `游릭` Minimap
- `游리` Extendable language support
- `游리` Find Replace
- `游리` Code Debugging
- `游리` Language Detection
- `游댮` Code Folding

## Usage
- Clone the repository and copy src folder to your project
- import cupcake to your script.

### Example: Basic Code Editor

- Run `python examples/basic.py`
- Open your script to edit
- Start editing.
