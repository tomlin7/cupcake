## Features

- Syntax highlighting
- Auto indentation
- Drag n drop

## Usage

### Prerequisites

- python
- pip packages
    > Install with `pip install -r requirements.txt`

### Running

- Run `python .` *(don't miss the dot in the end)*
- Open your cpp script.
- Start editing.
